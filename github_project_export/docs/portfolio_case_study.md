# Portfolio Case Study

This is a corrected Phase 1 RTO SPC dashboard using synthetic data.

The work demonstrates:

- RTO-only scope control
- Baseline-only per-stream limit calculation
- Thickness monitoring for RTR Mean, X-BAR, WIW Stdev, and SIGMA
- Particle threshold and repeated-event alerting
- Fleet overview charts that do not calculate shared limits
- Event persistence to `excursion_events.csv`
- Public-safe synthetic-data documentation

The dashboard is a portfolio demo and does not provide release decisions.
